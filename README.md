# NEW ME - Personal Growth & Motivation Website

## 🌟 Overview

**NEW ME** is an inspiring personal portfolio website dedicated to personal growth, self-reflection, and transformation. This project showcases a journey of overcoming failures and planning for success through web development, technology learning, and lifestyle improvements.

**Live Demo:** https://ahammad204.github.io/new-me/

---

## 🎯 Project Purpose

This website represents a commitment to personal development with three core areas of focus:
- **Web Development** - Building modern, responsive web applications
- **JavaScript** - Mastering the fundamentals of JavaScript programming
- **Lifestyle Improvement** - Breaking bad habits and maintaining a positive mindset

---

## ✨ Key Features

### 📱 Responsive Design
- Modern, clean UI with thoughtful color scheme
- Mobile-friendly layout with CSS flexbox
- Professional typography using Open Sans font family

### 📑 Main Sections

1. **Hero/Header Section**
   - Motivational headline: "All Thinks Are Possible If You believe"
   - Eye-catching banner background image
   - Call-to-action button

2. **Failure Analysis Section**
   - Reflection on previous year's challenges
   - Detailed list of lessons learned
   - Personal growth narrative with supporting imagery

3. **My Plan Section**
   - Four key goals with icon representations:
     - 🖥️ **Web Development** - Start learning and building
     - 📜 **JavaScript** - Master the language
     - 🚭 **Smoking** - End the habit
     - 🛒 **E-Commerce** - Develop a project

4. **Contact Section**
   - Motivational messaging
   - Direct contact call-to-action

5. **Footer**
   - About the future vision
   - Contact information
   - Social media links (Facebook, Twitter, YouTube)
   - Newsletter subscription form

---

## 🛠️ Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Professional styling and layout
  - Flexbox for responsive layouts
  - Custom color scheme with RGBA values
  - Box shadows and gradient effects
- **Images** - PNG icons and banner graphics

## 🚀 Getting Started

1. **Clone or Download the Repository**
   ```bash
   git clone https://github.com/ahammad204/new-me.git
   ```

2. **Open in Browser**
   - Simply open `index.html` in your web browser
   - Or access the live version: https://ahammad204.github.io/new-me/

3. **No Build Process Required**
   - Vanilla HTML, CSS, and static assets
   - Works directly in any modern browser

---

## 💡 How to Customize

### Update Content
- Edit `index.html` to change text, headlines, and contact information
- Modify styling in `style.css` for colors, fonts, and layout adjustments

### Replace Images
- Update banner images in `/images/banner/`
- Replace Icons in `/images/icons/`
- Update profile image `/images/person.png`

### Modify Colors
- Search for `rgba(0, 95, 172, 1)` in CSS to change the primary blue color
- Update text colors and backgrounds as needed
